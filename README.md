# Nastaran Esmaeil Zadeh - Portfolio Website

A modern, responsive portfolio website showcasing my work as an AI & Web Developer.

## About

This is my personal portfolio website featuring:
- Professional work experience and projects
- Academic publications and research
- Skills and certifications
- Contact information and downloadable CVs

## Features

- **Responsive Design** - Works on all devices (desktop, tablet, mobile)
- **Dark/Light Mode** - Toggle between themes
- **Interactive Galleries** - Project and certificate image carousels
- **Smooth Animations** - Modern UI with gradient effects
- **CV Downloads** - Both standard and Europass CV formats available

## Technologies Used

- HTML5
- CSS3 (with custom animations and gradients)
- JavaScript (vanilla)
- Google Fonts (Inter & Space Grotesk)

## How to Use

1. Clone this repository
2. Open `index.html` in your web browser
3. That's it! No build process or dependencies required.

## Sections

- **Home** - Introduction and quick links
- **About** - Background and education
- **Experience** - Work history and volunteering
- **Projects** - Academic and professional projects
- **Skills** - Technical and soft skills
- **Publications** - Research papers and articles
- **Certificates** - Professional certifications
- **References** - Professional contacts
- **Contact** - Get in touch
